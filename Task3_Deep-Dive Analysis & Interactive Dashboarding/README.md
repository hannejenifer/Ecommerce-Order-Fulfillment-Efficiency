# Task 3 – Deep Dive Analysis & Interactive Dashboarding

## Project Objective
The objective of this task was to perform deep analysis on the Olist e-commerce dataset and develop an interactive Power BI dashboard to track key business KPIs and operational performance.

## Key Analysis Performed

- Revenue trend analysis
- Order volume analysis
- Delivery performance analysis
- Customer distribution analysis
- Seller performance analysis
- Product category performance

## Key KPIs Developed

- Total Revenue
- Total Orders
- Average Order Value
- Average Delivery Days
- On-Time Delivery Rate
- Late Delivery Rate

## Dashboard Features

- Interactive filters (Year, State, Category)
- KPI summary cards
- Revenue trends
- Delivery performance metrics
- Customer and seller insights

## Tools Used

- Power BI (Dashboard Development)
- Python (Data Processing)
- Data Visualization
- Business Intelligence Concepts

## Key Business Insights

- Over 90% of deliveries were completed on time
- Late deliveries form a small but important segment affecting satisfaction
- Revenue is concentrated in top product categories
- Delivery performance is a key operational KPI

## Conclusion

This task strengthened my skills in KPI identification, dashboard design, and translating data into business insights.
